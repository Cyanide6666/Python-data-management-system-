# Python data management system: Sistema de Gestión de Datos de Empleados

## Descripción del proyecto:
Este sistema permite la manipulación de datos de manera eficiente utilizando las mejores prácticas de desarrollo.

## Características y Funcionalidades
El sistema cumple con los siguientes pilares de desarrollo:
- **Modularización:** Separación de la lógica de negocio (`gestion.py`) de la interfaz de usuario (`main.py`).
- **Estructuras de Datos:** Implementación de **Listas** para colecciones y **Diccionarios** para la representación de objetos de datos.
- **Recursividad:** Uso de funciones recursivas para realizar cálculos complejos (ej. sumatoria de salarios/costos).
- **Entrada/Salida de Datos:** Persistencia mediante lectura de archivos de texto (`.txt`) y salida formateada profesionalmente con **f-strings**.
- **Control de Flujo:** Estructuras condicionales y bucles robustos para la navegación en el sistema.

## Estructura del Repositorio
- main.py: Punto de entrada de la aplicación y manejo de menús.
- gestion.py: Módulo que contiene la lógica de procesamiento y funciones recursivas.
- datos.txt: Archivo fuente de datos para pruebas.
- docs/: Carpeta con la documentación técnica y manuales.
